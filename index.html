Chatbot de pré-diagnostic & gestion de lits

Doctor AI aide à décider si une visite à l'hôpital est nécessaire, oriente l'utilisateur vers l'établissement le plus proche et indique la disponibilité des lits en temps réel.

Voir hôpitaux disponibles

Langues supportées : Wolof et Français.

Problème identifié

Au Sénégal, les patients se déplacent souvent sans information fiable : ils ne savent pas s’ils doivent consulter et vers quel établissement se diriger. Cela provoque des surcharges dans les urgences et des délais de prise en charge.

Solution proposée

Doctor AI combine :

  • Un chatbot médical (IA + NLP) pour un pré-diagnostic.
  • Une API de cartographie pour la localisation.
  • Une base de données temps réel partagée par les hôpitaux pour la gestion des lits.

Techniques : modèles NLP légers, règles cliniques, API REST pour les hôpitaux, cache local pour la résilience réseau.

Impact attendu

  • Réduction des files d'attente
  • Mieux répartir les patients
  • Prévention via l'analyse des données (anticipation des pics)

Gestion des urgences

En cas de symptômes graves détectés par le chatbot, Doctor AI déclenche automatiquement une alerte hôpital :

  • Préparation de l’équipe médicale avant l’arrivée du patient.
  • Transmission des informations essentielles (symptômes, localisation, contact).
  • Orientation directe vers l’établissement le plus proche disposant de lits.
